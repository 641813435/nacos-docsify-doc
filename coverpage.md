# 微服务架构技术文档

> 基于[Spring Cloud Alibaba](https://github.com/alibaba/spring-cloud-alibaba/blob/master/README-zh.md)开发的微服务架构

- 基于敏捷开发SpringBoot2.1.x
- 集成分布式架构常见问题的主流解决方案
- 提供流程在线设计/管理平台
- 提供SOA架构、对WebService的两种处理方式
- 支持SSO CAS方式单点登录
- 支持大数据的简单处理
